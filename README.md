## Getting Started
**1. Install Dependencies:**
```
npm install
```

**2. apply config:**
```
yarn rn-nodeify --install url,events,https,http,util,stream,crypto,vm,buffer --hack --yarn
```

**3. Start iOS Simulator:**
```
react-native run-android
react-native run-android
```
